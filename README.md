# Biologo.fotos - Portafolio de Fotografía

Aplicación web para mostrar y gestionar un portafolio de fotografía, con galería dinámica y posibilidad de agregar nuevas imágenes desde el frontend.

---

## Tecnologías

- Frontend: React, Create React App, CSS
- Backend: Node.js + Express (para subir y servir imágenes)
- Base de datos: MongoDB
- Herramientas: ESLint, Git
- Despliegue: Vercel

---

## Funcionalidades

- Visualización de galería de fotos por categoría
- Subida de nuevas fotos con título desde el frontend
- Slider automático de imágenes
- Interfaz responsive y amigable
- Mock backend disponible para desarrollo local
